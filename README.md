# GUISolo

**GUISolo** is a Java Swing application that lets you interactively assign stat points to a character, inspired by RPG-style leveling systems such as *Solo Leveling*. The program provides a graphical interface to manage a character's attributes like INT, STR, DUR, etc., using buttons and live updates.
